METHODS
-------
1. forecast_cashflows(base_cashflow, growth_rate, years)
   Returns list of CFs for periods 1..years. NOT rounded internally —
   rounding the year-3 CF before terminal value causes a 0.87 error
   that cascades into a wrong share price.

2. present_value(cashflow, discount_rate, period)
   CF / (1 + r)^t  →  round(result, 2)

3. terminal_value(last_forecasted_cf, growth_rate, discount_rate)
   Gordon Growth: last_cf × (1 + g) / (r - g)  →  round(result, 2)
   Raises ValueError if discount_rate <= growth_rate.

4. pv_of_terminal_value(terminal_value, discount_rate, periods)
   TV / (1 + r)^n  →  round(result, 2)

5. intrinsic_equity_value(forecasted_fcfe, ke, terminal_growth_rate)
   Sum of PV of all FCFEs + PV of terminal value.
   Full precision internally, rounds only the final total.

6. intrinsic_share_price(equity_value, shares_outstanding)
   equity_value / shares_outstanding  →  round(result, 2)
   Raises ValueError if shares_outstanding is zero.
   NOTE: units must match — equity value is in crore INR, so pass
   shares in crore (807617120 / 10_000_000 = 80.761712).

7. intrinsic_enterprise_value(forecasted_fcff, wacc, terminal_growth_rate)
   Same as intrinsic_equity_value but uses FCFF discounted at WACC.

8. valuation_verdict(intrinsic_price, market_price, margin=0.10)
   intrinsic > market × 1.10  →  "Undervalued"
   intrinsic < market × 0.90  →  "Overvalued"
   otherwise                  →  "Fairly Valued"

9. sensitivity_analysis(base_fcfe, ke_values, tg_values, shares_outstanding, growth_rate, years)
   Returns 2D dict {ke: {tg: share_price}} for every (ke, tg) combination.
   Cells where ke <= tg return None (Gordon Growth undefined).
